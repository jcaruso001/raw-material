The More You Know

When comparing financial aid and packages offered by each college, make sure you read the fine print. While one college might appear to be offering more funding, your out-of-pocket expenses for that college might actually be higher. Also, make sure you clearly understand what each aspect of the package is and what monies must be repaid.

Solving Quadratic Equations by Factoring

Don’t be surprised if you are asked to solve quadratic equations that can be factored. You will not
be asked to use the quadratic formula to solve these equations.

Example 1: If x^2+ 5x + 6 = 0, what are the two possible values of x?

First, we need to put the quadratic into factored form. To do this we ask ourselves, what two numbers multiply together to get 6 and add together to get 5? The answer is 3 and 2 so the factored for is:

(x + 3)(x + 2) = 0

We can see that the product of the two factors is equal to zero. Since the only way to have a product equal to zero is to have multiplied by zero, this implies

(x + 3) = 0 and/or (x + 2) = 0

Solving for x gives us the two solutions, x = -3 and x = -2.

Example 2: If x^2 + 12 = 7x ,what are the two possible values of x?

First, subtract 7x from both sides to get the standard quadratic equation.

x^2– 7x + 12 = 0
(x – 4)(x – 3) = 0

Therefore, (x – 4) = 0 and/or (x – 3) = 0. Solving each equations for s gives the solutions x = 4 and x = 3.

Example 3: How many different solutions are there for the equation?

2x + 6 = (x + 5)(x + 3)?

A) 0
B) 1
C) 2
D) 3

First multiply the binomials.

2x + 6 = x^2 + 8x + 15

Then get all the terms on one side, then combine like terms in order to get the standard quadratic equation.

x^2 + 6x + 9 = 0
(x + 3)(x + 3) = 0

Solving for x gives a single solution of – 3. Therefore, the answer to the question is B since there is only one solution, – 3.

Chapter Four: Solving Systems of Linear Equations

Solving Systems of Linear Equations

You will be asked to solve systems of two or more linear equations or inequalities. Linear systems are equations that contain the same variables. Thus, a + 2b = 11 and 2a + b =10 are linear systems since they both contain the same variables, a and b.

To solve systems of linear equations, you can use the substitution method:
  • Take one of the equations listed and solve for one of one of the variables in terms of the other.
  • Substitute the value found for the variable into the other equation.
  • Solve for the second variable.
  • Substitute the value of the second variable into the original equation to solve for the value of the first variable.


Example: For what values of x and y are the following equations true?

2x + 4y = 50
3x + 5y = 66

In the first equation, solve for x in terms of y.

2x + 4y = 50
2x = 50 – 4y
x = 25 – 2y

Then substitute for x in the other equation and solve for y.

3x + 5y = 66
3(25 – 2y) + 5y = 66
75 – 6y + 5y = 66
75 – y = 66
– y = –9
y = 9

Substitute y = 9 back into the original equation, 2x + 4y = 50, in order to find the value of x.

2x +4(9) = 50
2x +36 =50
2x = 14
x = 7

In some cases, using substitution can be difficult. If neither equation can be easily solved for one of the variables, use elimination instead.
  • Pick the variable you want to eliminate.
  • Multiply each equation by the quantity needed so that the coefficients of the variable to be eliminated are opposite in value.
  • Add the equations together so that the variable to be eliminated is canceled out
  • Solve for the second variable.
  • Substitute the value of the second variable into one of the original equations to solve for the value of the first variable.

Example: For what values of x and y are the following equations true?

2x + 5y = 19
3x + 2y = 12

Solving either equation for x or y would result in fractions so let’s eliminate instead. We can eliminate x by multiplying the first equation by 3 and the second equation by -2.

6x + 15y = 57
-6x – 4y = -24

Now we add the equations together. The two x terms will cancel out.

11y = 33
y = 3

Now we substitute 3 for y into one of the original equations and solve for x. Let’s substitute in the first equation.

2x + 5(3) = 19
2x + 15 = 19
2x = 4
x = 4

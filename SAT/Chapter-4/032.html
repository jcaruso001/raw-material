Extraneous Solutions

When solving radical equations, we sometimes get solutions that do not actually work when plugged back into the equation. This is because in order to solve the radical equation, we squared both sides. Whenever a negative number is squared, it becomes positive which can affect the solution. To avoid getting the incorrect answer, always plug the solutions back into the original equation to verify.

Example: 

First, we square both sides.


We have two solutions, x = 1 and x = 6. We need to plug both of these into the original equation to make sure they are both correct.


When we plug 1 into the original equation, we get the false statement, √4=-2. Thus 1 is not a solution. When we plug 6 into the original equation, we get the true statement, √9=3. Thus 6 is a solution. Therefore, there is only one solution to this radical equation and that solution is x = 6.

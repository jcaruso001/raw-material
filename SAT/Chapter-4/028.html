Finding the Equation of a Parallel and Perpendicular Line

Example: Find the equation of the line parallel to the line y = 2x −1 that contains the point (5, 6).

As always, we start by writing down the formula for a line

y = mx + b

We need to find the values of m and b. Since our line is parallel to y = 2x −1, the lines will have the same slope. The slope of y = 2x −1 is 2, so m = 2.

y = 2x + b

Now we can solve for b using the fact that our line contains the point (5, 6). Thus, when x = 5, y = 6.

>6 = 2(5) + b
6 = 10 + b
−4 = b

Thus, the equation of the line parallel to the line y = 2x −1 that contains the point (5, 6) is

y = 2x −4

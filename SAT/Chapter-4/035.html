Functions

A function is a rule or formula that tells how to relate elements in one set (the domain) with the elements in another set (the range). There are many different types of function notations that you should be familiar with. Here are a few examples:


In each case, a specific value of x will give you a specific value of y, f(x), or g(x).

Example 1: Given f(x) = x2 + 2x − 1, find f(2).

When you see a question like this, all you need to do is substitute the new x value into the function. In this case, substitute 2 for x in the function.
